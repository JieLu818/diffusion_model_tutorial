# 条件扩散模型

> **一句话总结**：无条件扩散模型只能随机生成数字（你不知道会生成"3"还是"7"）。条件扩散模型让生成过程受控制——你告诉它"给我生成一个'8'"。

## 从无条件到有条件

### 无条件生成（Unconditional）

我们目前实现的扩散模型是**无条件**的：

$$p_\theta(x_{t-1} | x_t)$$

网络只知道输入噪声图和时间步，不知道要生成什么数字。结果就是随机生成 0-9 中的任意数字。

### 条件生成（Conditional）

条件扩散模型在输入中加入**条件信息** $c$：

$$p_\theta(x_{t-1} | x_t, c)$$

其中 $c$ 可以是：
- **类别标签**（比如数字"5"）
- **文本描述**（比如"一只猫"）
- **另一张图**（比如超分辨率、图像修复）
- **任何你想控制的东西**

> **大白话**：无条件像"随机画画"，有条件像"照着命题画画"。

## 实现类别条件扩散

### 方法 1：条件注入（最简单）

把类别信息也编码成向量，加到 U-Net 的每一层。

```python
class ClassConditionedUNet(UNet):
    """在 U-Net 中加入类别条件"""
    def __init__(self, config, num_classes=10):
        super().__init__(config)
        
        # 类别嵌入层：把 label（0-9）映射成向量
        self.class_embedding = nn.Embedding(num_classes, config.time_emb_dim)
    
    def forward(self, x, t, labels=None):
        # 时间嵌入
        t_emb = self.time_embed(t)
        
        # 如果给了类别条件，把类别嵌入加到时间嵌入上
        if labels is not None:
            c_emb = self.class_embedding(labels)
            t_emb = t_emb + c_emb  # 条件和时间信息融合
        
        # 其余 forward 不变...
```

**关键改动**：
1. 加入 `nn.Embedding(num_classes, time_emb_dim)` 把类别标签映射成向量
2. 将类别向量加到时间嵌入上（或者在任何位置拼接/相加）
3. forward 函数多接收一个 `labels` 参数

> **大白话**：类别条件的实现其实非常简单——就是把"我想生成数字 8"这个信息，通过相加的方式告诉网络的每一层。

### 方法 2：Classifier-Free Guidance（CFG）

CFG 是目前最主流的条件扩散方法（Stable Diffusion 用的就是它）。

**核心思想**：同时训练一个无条件模型和一个条件模型，用两者的组合来生成。

**训练时**：以一定概率（如 10%）丢弃条件信息，让网络学会无条件和有条件两种模式。

```python
# 训练时：以 10% 概率丢弃条件
if random.random() < 0.1:
    labels = None  # 无条件模式
else:
    labels = labels  # 有条件模式

pred_noise = model(x_t, t, labels)
```

**采样时**：用"引导强度" $w$ 控制条件影响：

$$\tilde\epsilon = \underbrace{\epsilon_\theta(x_t, t, \emptyset)}_{\text{无条件}} + w \cdot \left(\underbrace{\epsilon_\theta(x_t, t, c)}_{\text{有条件}} - \underbrace{\epsilon_\theta(x_t, t, \emptyset)}_{\text{无条件}}\right)$$

> **大白话**：先算"无条件的噪声预测"，再算"有条件的噪声预测"，然后用两者的差值调整生成方向。$w$ 越大，生成的图像越符合给定条件。

## 条件扩散的应用

### 1. 文本到图像（Text-to-Image）

这就是 Stable Diffusion、DALL-E 做的事情。

条件 $c$ = 文本描述，用 CLIP 或其他文本编码器提取特征。

### 2. 图像修复（Inpainting）

给定一张缺了一块的图片，让扩散模型填充缺失部分。

条件 $c$ = 已知像素区域，只对未知区域做去噪。

### 3. 超分辨率（Super-Resolution）

给定一张低分辨率图片，生成高分辨率版本。

条件 $c$ = 低分辨率图片（通常先上采样到目标尺寸，然后作为额外通道输入）。

### 4. 可控生成

- **ControlNet**：在 U-Net 的编码器中加入额外条件（边缘图、姿态图、深度图等）
- **LoRA**：用小量参数微调，生成特定风格

## 要点回顾

1. **条件扩散** = 在输入中加入条件信息 $c$
2. **类别条件**可以用 `nn.Embedding` + 和时间嵌入相加实现
3. **Classifier-Free Guidance** 用无条件 + 有条件预测的差值控制生成强度
4. 条件扩散是**文字到图像、图像修复、超分**等应用的基础
5. Stable Diffusion 等主流模型都基于条件扩散
