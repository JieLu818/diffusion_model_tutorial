# 训练循环与采样流程

> **一句话总结**：本章将调度器、U-Net 和扩散流程组装起来，实现训练循环（让网络学会去噪）和采样流程（从噪声生成图像）。

## 扩散流程管理类

```python
class DiffusionPipeline:
    """
    组装 model + scheduler，提供：
    - train_step：单步训练
    - sample：完整采样生成
    """
    def __init__(self, model, scheduler, config, device="cpu"):
        self.model = model
        self.scheduler = scheduler
        self.config = config
        self.device = device
```

## 1. 训练一步（train_step）

```python
def train_step(self, x_0, optimizer):
    """
    单步训练
    
    1. 随机选时间步 t
    2. 对 x_0 加噪得到 x_t
    3. U-Net 预测噪声
    4. 计算 MSE 损失
    5. 反向传播
    """
    B = x_0.shape[0]
    
    # (1) 随机采样时间步：每个样本随机选一个 t
    t = torch.randint(0, self.scheduler.T, (B,), device=self.device)
    
    # (2) 前向加噪：直接从 x_0 跳到 x_t
    noise = torch.randn_like(x_0)
    x_t, _ = self.scheduler.q_sample(x_0, t, noise)
    
    # (3) U-Net 预测噪声
    pred_noise = self.model(x_t, t)
    
    # (4) 计算 MSE 损失：L = MSE(ε, ε_θ(x_t, t))
    loss = F.mse_loss(pred_noise, noise)
    
    # (5) 反向传播
    optimizer.zero_grad()
    loss.backward()
    torch.nn.utils.clip_grad_norm_(self.model.parameters(), max_norm=1.0)
    optimizer.step()
    
    return loss.item()
```

**为什么只需要一步？**
因为前向加噪有跳跃公式：$x_t = \sqrt{\bar\alpha_t} x_0 + \sqrt{1-\bar\alpha_t} \epsilon$。不需要从 $t=0$ 一步步走到 $t=T$。

**为什么每个样本 t 不同？**
随机选 t 保证网络在所有时间步上都能学习到去噪。

> **大白话**：训练时，网络就像在做"噪声盲猜"——给它一张加了随机量噪声的图，让它猜这个噪声长什么样。每次加噪量都不同，从几乎没加到几乎全是噪声，网络都要会。

## 2. 采样（sample）— 生成新图像

```python
@torch.no_grad()
def sample(self, batch_size=16):
    """
    从纯噪声开始，逐步去噪生成新图像
    
    算法：
    x_T ~ N(0, I)
    for t = T-1, ..., 0:
        ε = model(x_t, t)                    // 预测噪声
        x_{t-1} = 1/√α_t · (x_t - β_t/√(1-ᾱ_t) · ε)
               + √β_t · z                    // (t>0 时加随机噪声)
    """
    self.model.eval()
    
    # (1) 从纯噪声开始
    x = torch.randn(batch_size, 1, 28, 28, device=self.device)
    
    # (2) 逐步去噪
    for t_step in range(self.scheduler.T - 1, -1, -1):
        t = torch.full((batch_size,), t_step, device=self.device, dtype=torch.long)
        
        # 预测噪声
        pred_noise = self.model(x, t)
        
        # 获取去噪系数
        coeff1, coeff2 = self.scheduler.get_pred_mean_coeffs(x, t, pred_noise)
        
        # 计算 x_{t-1} 的均值
        pred_mean = coeff1 * (x - coeff2 * pred_noise)
        
        # 加随机噪声（最后一步不加）
        if t_step > 0:
            variance = self.scheduler.get_variance(t_step)
            noise = torch.randn_like(x)
            x = pred_mean + torch.sqrt(variance) * noise
        else:
            x = pred_mean
    
    self.model.train()
    return torch.clamp(x, -1.0, 1.0)
```

**采样为什么比训练慢这么多？**

训练：随机选一个 t → 一步算出 x_t → 一步算损失 → 完成。
采样：必须从 T 一步步走到 0 → T 步（比如 1000 步）。

| | 训练 | 采样 |
|---|---|---|
| 前向传播次数 | 1 次 | T 次（如 1000 次） |
| 相对时间 | 1 倍 | 1000 倍 |

## 要点回顾

1. `train_step` 实现**单步训练**：随机选 t → 加噪 → 预测噪声 → MSE 损失
2. `sample` 实现**完整生成**：从纯噪声开始，走 T 步逐步去噪
3. 训练**快**：每个样本只做 1 次前向
4. 采样**慢**：必须走完 T 步（1000 次前向）
5. 梯度裁剪（`clip_grad_norm_`）防止梯度爆炸
6. 训练完成后模型保存在 `output/` 目录下
