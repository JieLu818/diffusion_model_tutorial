# 前向加噪与数据集准备

> **一句话总结**：本章实现数据加载和可视化前向加噪过程，让你亲眼看到一张 MNIST 数字图如何被逐渐噪声淹没。

## 数据准备

### MNIST 数据集

MNIST 是"Hello World"级别的图像数据集：28×28 的灰度手写数字（0-9），共 60000 张训练图 + 10000 张测试图。

我们用它来训练扩散模型。

### 数据预处理

```python
from torch.utils.data import DataLoader
from torchvision import datasets, transforms

transform = transforms.Compose([
    transforms.ToTensor(),  # [0, 1]   → PIL 图像转 Tensor
    transforms.Lambda(lambda x: x * 2 - 1),  # [-1, 1] → 扩散模型需要
])

dataset = datasets.MNIST(root="./data", train=True, download=True, transform=transform)
loader = DataLoader(dataset, batch_size=128, shuffle=True)
```

**关键点**：像素值映射到 $[-1, 1]$ 范围，和噪声 $\epsilon \sim \mathcal{N}(0, 1)$ 同一个量级。如果不做这个映射，信号的幅度和噪声的幅度不匹配，训练会不稳定。

> **大白话**：把像素从 [0,1] 压缩到 [-1,1]，这样信号（图像像素）和噪声（从 N(0,1) 采样的值）都在差不多的数值范围，网络更容易学习。

## 前向加噪的可视化

来看一下实际的加噪效果。我们取一张 MNIST 数字"8"，看看它在不同噪声步的表现：

![[../images/forward_process_illustration.png]]

图中展示了 $t=0$（原始）到 $t=999$（纯噪声）的图像变化过程：

| 时间步 t | $\bar\alpha_t$ | $\sqrt{1-\bar\alpha_t}$ | 可见效果 |
|---|---|---|---|
| 0 | 1.0000 | 0.0000 | 原始图，完全清晰 |
| 100 | 0.9886 | 0.1068 | 几乎没变化，加了一点点噪点 |
| 300 | 0.9094 | 0.3011 | 开始出现可见噪声，但数字仍清晰 |
| 500 | 0.6114 | 0.6233 | 噪声和信号各占一半，数字模糊 |
| 700 | 0.1458 | 0.9243 | 噪声占主导，只有隐约轮廓 |
| 900 | 0.0053 | 0.9973 | 几乎完全被噪声淹没 |
| 999 | 0.0001 | 0.9999 | 纯噪声，看不出任何结构 |

> **大白话**：$\bar\alpha_t$ 从 1 降到接近 0，意味着原始信息一步步被噪声侵蚀。$\sqrt{1-\bar\alpha_t}$ 从 0 上升到接近 1，意味着噪声一步步增强。

## 在训练脚本中

我们在 `train_mnist.py` 中包含了前向加噪的可视化功能：

```python
def save_forward_process(pipeline, dataset, save_path, num_steps=10):
    """可视化前向加噪过程"""
    x_0 = dataset[0][0].unsqueeze(0).to(device)
    
    T = scheduler.T
    step_size = T // (num_steps - 1)
    all_steps = [0] + [i * step_size for i in range(1, num_steps)]
    
    fig, axes = plt.subplots(1, num_steps, figsize=(num_steps * 1.5, 2))
    for i, t_val in enumerate(all_steps):
        t = torch.full((1,), t_val, device=device, dtype=torch.long)
        x_t, _ = scheduler.q_sample(x_0, t)
        img = (x_t[0].cpu() + 1) / 2  # 从 [-1, 1] 映射回 [0, 1]
        axes[i].imshow(img.squeeze(), cmap="gray")
        axes[i].set_title(f"t={t_val}")
        axes[i].axis("off")
    plt.savefig(save_path, dpi=150)
```

## 要点回顾

1. 数据需要**归一化到 [-1, 1]**，和噪声同量级
2. $\bar\alpha_t$ 从 1 → 0，对应原始信息从**全部保留**到**完全丧失**
3. $\sqrt{1-\bar\alpha_t}$ 从 0 → 1，对应噪声从**无**到**主导**
4. 在 $t \approx 500$ 时，噪声和信号约各占一半
5. 可视化前向过程有助于**直观理解**网络在不同时间步看到的是什么
